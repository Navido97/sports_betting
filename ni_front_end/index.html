Cloud Architecture

Diagram coming soon. Architecture includes S3 for storage, EC2 for hosting, GitHub for version control, and Tableau Public for dashboards.

How It Works

Data is stored in S3 and updated via Python scripts. Tableau connects to this data and updates dashboards. GitHub hosts the HTML files, which are served via EC2. Updates are synced automatically or manually.
