Welcome to the Odds Maker Dashboard

This dashboard helps everyday sports fans and bettors stay up to date with betting odds for international soccer competitions. Simply select a tournament from the left-hand menu to view odds from major sportsbooks for upcoming and past matches. Use this tool to compare values across platforms and make informed betting or viewing decisions.

Step-by-Step Breakdown

  1. Define Project Scope: The goal was to create a free, automated dashboard showing real-time soccer betting odds from multiple competitions.
  2. Fetch Odds via API: Used the-odds-api.com for its free-tier access to sports betting data. Python scripts fetch odds for World Cup, Club World Cup, and Gold Cup daily.
  3. Store Data in AWS S3: Chose S3 due to its free-tier 5GB storage and ease of integration with Python via Boto3. Each dataset was saved as a CSV to the S3 bucket.
  4. Bridge to Tableau via Google Sheets: Since Tableau Public cannot pull from S3 directly, I used the Google Sheets API to push each CSV to a unique tab in one Google Sheet. This allowed live access to the data.
  5. Build Dashboards in Tableau Public: Tableau Public is free and powerful. Each sheet is linked to a Google Sheets tab and visualized with relevant filters, team names, and odds spread metrics.
  6. Design a Lightweight Front-End: I built the front-end using plain HTML, CSS, and JS with tab navigation and sidebar menu. Each tab contains a Tableau dashboard embed and associated context.
  7. Automate on EC2 (Free Tier): The Python pipeline runs daily on a t2.micro EC2 instance. This instance also serves the front-end files using a basic static HTTP server.
  8. Document the Project: Finally, I created a high-level chart and added this section to document the steps, tools, and decisions — all designed to stay within the free-tier ecosystem.

This project showcases how a fully automated, data-driven website can be built and hosted end-to-end using only free-tier tools — without sacrificing quality or automation.

The website was built using a lightweight front-end architecture: HTML, CSS, and vanilla JavaScript. The layout is structured around a sidebar menu with tabs dynamically shown and hidden via JavaScript. Tableau dashboards are embedded directly through the public API. All betting data is fetched through scheduled Python scripts and stored on AWS S3, which is then surfaced in Tableau through a connected Google Sheet.

World Cup Winners

This dashboard displays outright winner odds for the FIFA World Cup. You can compare which countries are favored to win the tournament, with odds pulled from multiple betting platforms. Use it to track line movement and identify longshot opportunities.

Concacaf Gold Cup

This dashboard shows match-by-match betting odds for the Concacaf Gold Cup. Each matchup includes odds from multiple sportsbooks, enabling you to find the best line for moneyline bets across North American teams.

Club World Cup

This dashboard provides odds for the FIFA Club World Cup matches, showing bookmaker spreads and predictions for games between top clubs from each continent. Useful for comparing teams that don't typically meet in domestic leagues.
